# Brendanhu.com website

## Setup
 * `git clone git@github.com:mithridatize/brendanhu.com.git;`
 * `gem install bundler;`
 * `rbenv local 2.6.3;`
 * `bundle install;`

## Local Development
`bundle exec jekyll serve;`

## Deployments
Just push to Github.
Deployments happen automagically via [Github Pages](https://pages.github.com/).
